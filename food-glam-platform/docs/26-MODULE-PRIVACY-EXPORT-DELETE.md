# Module: Privacy + Data Portability + Delete

## Purpose
Give users control: export their data and delete it.

## Features
- Export to JSON/CSV:
  - meal plans
  - shopping lists
  - logs (weight, food, water)
  - pantry
- Delete account (hard delete) or deactivate

## Routes
- `/me/settings/privacy`

## Acceptance criteria
- Data export works.
- Delete account deletes private data.
